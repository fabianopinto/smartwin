# Contributing to SmartWin

Thank you for your interest in contributing to SmartWin!

## How to contribute

- Fork the repository.
- Create a feature branch: `git checkout -b feature-name`.
- Commit your changes with clear messages.
- Open a pull request against the `main` branch.

## Reporting issues

Please open an issue for bugs, feature requests, or unexpected behavior.

## Code style

- Keep Swift code clean and consistent.
- Prefer descriptive variable and function names.
- Add comments only when necessary.
